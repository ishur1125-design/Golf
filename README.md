# ⛳ 골프 내기 정산

홀별 타수 내기를 계산해 주는 웹앱입니다. 스코어만 넣으면 **누가 누구에게 얼마를 줘야 하는지**가 바로 나옵니다.

👉 **[앱 열기](https://ishur1125-design.github.io/Golf/)**

설치할 것 없이 브라우저에서 바로 쓰고, 입력한 내용은 그 기기에 자동 저장됩니다. 서버로 아무것도 보내지 않습니다.

## 무엇을 하나

- **1:1 전원 비교 정산** — 홀마다 모든 짝을 비교해 `타수차 × 타당금액`을 주고받습니다. 합계는 항상 정확히 0원입니다.
- **두 가지 시각** — 실제로 발생한 금액을 보여주는 **총액 매트릭스**와, 최소 송금 건수로 끝내는 **순액 정산**을 함께 봅니다.
- **배판 자동 판정** — 트리플 이상(파3는 더블 이상) · 버디 이상 · 3명 동타(4인 이상) · 3퍼트 · OB · 해저드. 조건별로 켜고 끌 수 있고, 걸린 이유가 화면에 표시됩니다. 수동 지정·해제도 됩니다.
- **버디값 · 이글값** — 언더파를 친 사람이 못 친 사람 각자에게서 따로 받습니다. 홀 금액에 연동(배판 반영)하거나 고정 금액으로 설정할 수 있고, 이글값은 버디값과 별도로 정합니다.
- **페널티 별도 금액** — OB, 해저드, 기타 벌타를 각각 다른 금액으로 설정합니다.
- **라운드 중 금액 변경** — 아무 홀에서나 타당금액을 올리거나 내리고, 남은 홀 전체에 한 번에 적용합니다.
- **현재 판세** — 지금까지 각자 얼마를 따고 잃었는지 스코어 화면에 계속 표시됩니다.
- **전반 / 후반** — 1~9번씩 나눠 보고, 정산도 전반·후반 소계로 나옵니다.
- **1~5인**, 9홀 / 18홀
- **저장과 공유** — 라운드를 기기에 저장하고, 정산표를 카카오톡에 붙여넣을 텍스트로 복사합니다.

## 파일

| 파일 | 설명 |
|---|---|
| `index.html` | 실제로 서비스되는 페이지. 이 파일 하나가 앱 전부입니다. |
| `app.html` | 본문 소스 (`<head>` 없는 형태) |
| `head.part` | `index.html`을 만들 때 앞에 붙는 `<head>` |
| `build.sh` | `app.html` + `head.part` → `index.html` |

`app.html`을 고쳤다면 `./build.sh`를 실행해 `index.html`을 다시 만든 뒤 커밋하세요. `index.html`만 직접 고쳐도 앱은 잘 돌아갑니다.

## 정산 규칙 예시

파4, 타당 ₩1,000, 네 명이 각각 4·5·6·4타를 쳤을 때:

| | A | B | C | D |
|---|---|---|---|---|
| 순액 | +3,000 | −1,000 | −5,000 | +3,000 |

A는 B에게 1타(₩1,000), C에게 2타(₩2,000)를 받고 D와는 비깁니다. 나머지 짝도 같은 방식으로 계산되며 네 명의 합은 0원입니다.
